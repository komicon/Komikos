# KomikOS · iOS 26 Style
Web baca komik **Manga · Manhwa · Manhua Bahasa Indonesia**, desain mirip iOS 26 Liquid Glass, **100% mobile friendly**, bisa update otomatis, terintegrasi penuh GitHub.

## ✨ Fitur
- 🔥 Slider otomatis komik populer
- 🎨 Tema terang/gelap otomatis iOS
- ⚙️ Panel admin CRUD + push GitHub
- ⏰ Update otomatis tiap jam via Actions
- 📱 Navigasi bawah melayang, aman Dynamic Island

## 🚀 Pasang
1. Upload semua ke repo GitHub
2. **Settings → Pages → Source = GitHub Actions**
3. Buka `https://NAMAKAMU.github.io/komik-ios26/`
4. Admin: `/admin.html` → `admin / admin123`
